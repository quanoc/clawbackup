🎉 欢迎使用 Quinoa 主题

这是一个基于 Hexo 的现代化知识管理主题。

✨ 主要特性

  • 双模式路由 - SPA 无刷新导航 + 独立页面分享
  • 层级分类 - 侧边栏多级分类树
  • 全文搜索 - 闪电般的客户端搜索
  • 响应式设计 - 完美适配桌面、平板、手机
  • AI 就绪 - Markdown 原生，适合 AI 生成内容

📝 写作指南

  1. 创建新文章：hexo new "文章标题"
  2. 指定分类：在 front-matter 中添加 categories:
  3. 生成静态文件：hexo generate
  4. 本地预览：hexo server

🔧 下一步

  • 编辑 themes/Quinoa/_config.yml 自定义主题
  • 编辑 _config.yml 配置站点信息
  • 开始写作你的知识文章！

祝你使用愉快！ 📚
