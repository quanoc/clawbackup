## 🚀 SEO 优化建议（按优先级）

### 🔴 P0 优先级（本周完成）

#### 1. 添加 Schema.org 结构化数据

**作用**: 让 Google 显示富摘要（作者、时间、评分等）

**实施**: 在 `themes/quinoa/layout/_partial/head.ejs` 添加：

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "name": "Nova's Wiki",
  "url": "https://waisec.cn",
  "description": "Quinoa 主题驱动的个人知识库，记录技术学习与思考",
  "author": {
    "@type": "Person",
    "name": "Nova"
  },
  "datePublished": "2026-03-07",
  "dateModified": "2026-03-07"
}
</script>
```

**预期效果**: 
- Google 搜索结果显示作者信息
- 点击率提升 15-25%

**耗时**: 30 分钟

---

#### 2. 优化文章页面 og:type

**当前**: 所有页面都是 `website`  
**建议**: 文章页使用 `article`

**实施**: 编辑主题模板

```ejs
<% if (is_post()) { %>
  <meta property="og:type" content="article">
  <meta property="article:author" content="<%= config.author %>">
  <meta property="article:published_time" content="<%= page.date.toISOString() %>">
<% } else { %>
  <meta property="og:type" content="website">
<% } %>
```

**预期效果**:
- 社交媒体分享更专业
- 分享点击率提升 30%

**耗时**: 30 分钟

---

#### 3. 启用 RSS/Atom 订阅

**作用**: 提升用户粘性，方便订阅更新

**实施**:
```bash
npm install hexo-generator-feed --save
```

```yaml
# _config.yml
feed:
  type: atom
  path: atom.xml
  limit: 20
  order_by: -date
```

**预期效果**:
- 提升回头客比例
- 方便内容聚合平台抓取

**耗时**: 15 分钟
